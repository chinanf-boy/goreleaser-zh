Checksum-校验

GoReleaser生成一个project_1.0.0_checksums.txt文件，并在发布时上传,以便您的用户可以验证下载的文件是否正确.

该checksum部分允许自定义文件名:

# .goreleaser.yml
checksum:
  # 选择 checksums 的名称.
  # 默认为 `{{ .ProjectName }}_{{ .Version }}_checksums.txt`.
  name_template: "{{ .ProjectName }}_checksums.txt"
